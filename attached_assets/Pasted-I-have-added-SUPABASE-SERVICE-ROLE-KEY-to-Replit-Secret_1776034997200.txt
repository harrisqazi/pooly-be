=== METHODOLOGY — SELF HEALING TEST RUNNER ===

GENERAL PHILOSOPHY:
This script is a self-healing autonomous test runner.
It must never give up on a test just because the first
fix did not work. It must think, learn from its own
history, and try progressively different approaches.
It only stops when it has genuinely exhausted all
reasonable options and documents exactly why.

FIX PHASE RULES:
- If any database fix fails, do not proceed to tests.
- Log the exact error, diagnose it, attempt to fix the
  fix itself (e.g. wrong column name, RLS blocking,
  type mismatch), retry the fix up to 5 times.
- Only stop the entire script if the fix cannot be
  resolved after 5 attempts with different approaches.
- Before stopping always write to test-log.md with
  full diagnosis and recommended manual steps.

TEST PHASE RULES — PER TEST:
Each test gets up to 5 attempts total.
Between each attempt the script must:
  1. Read the exact error from the response
  2. Check fixHistory for this test — never repeat
     the same fix twice
  3. Diagnose using the predicted causes list first
  4. If error does not match any predicted cause:
     CATCH-ALL DIAGNOSIS MODE:
       - Log: "Unknown failure — entering catch-all diagnosis"
       - Read the relevant route file in full
       - Read the relevant middleware in full
       - Query Supabase directly to check data state
       - Check server logs via console output
       - Form a hypothesis about what is wrong
       - Log the hypothesis clearly
       - Attempt the fix based on the hypothesis
       - Log exactly what was changed
  5. After each fix attempt:
     - Create a checkpoint with label and timestamp
     - Log the fix in test-log.md under that test
     - Restart the server if a route file was changed
     - Wait 3 seconds for server to restart
     - Confirm server is up by hitting GET /health
     - Retry the test
  6. If after 5 attempts the test still fails:
     - Log "EXHAUSTED 5 ATTEMPTS"
     - Write full history of all 5 attempts to test-log.md
       including what was tried, what the error was each
       time, and why each fix did or did not work
     - Write a RECOMMENDED MANUAL FIX section in the log
     - Create a checkpoint labeled "stopped-at-test-N-[timestamp]"
     - STOP the script entirely
     - Do not proceed to subsequent tests

SELF HEALING HISTORY:
Maintain an in-memory object called fixHistory:
  fixHistory = {
    'test-N': [
      {
        attempt: 1,
        error: 'exact error message',
        hypothesis: 'what we thought was wrong',
        fixApplied: 'what we changed',
        result: 'pass/fail',
        timestamp: 'ISO string'
      }
    ]
  }

Before each fix attempt, read fixHistory for that test.
Never apply the same fix twice.
Use the history to inform the next hypothesis —
if fix A failed, reason about why and try fix B.
The history is the script's memory — use it.

CATCH-ALL DIAGNOSIS CHECKLIST:
When error does not match predicted causes,
check these in order and log findings for each:
  1. Is the server running and reachable? (GET /health)
  2. Is the auth header being sent correctly?
  3. Does the route exist? (check server.js mounts)
  4. Is the request body formatted correctly?
  5. Is the database row in the expected state?
     (query Supabase directly and log the result)
  6. Are there any foreign key violations?
  7. Are column names correct vs schema?
  8. Is middleware blocking before route handler?
  9. Is there a JavaScript runtime error in the route?
     (check if similar requests work with minimal body)
  10. Is idempotency returning a cached wrong response?
      (try with a brand new idempotency_key)

SERVER RESTART PROCEDURE:
When a route file is modified:
  1. The server is already running as a background process
  2. Find the node process PID and send SIGTERM
     or use: pkill -f 'node server.js' then restart
  3. Wait 3 seconds after restart
  4. Confirm server is up by hitting GET /health
  5. If /health fails after restart:
     log server crash, read error, fix startup issue first
     do not run the test until server is confirmed up

function log(msg) {
  const line = typeof msg === 'object'
    ? JSON.stringify(msg, null, 2) : msg
  console.log(line)
  logBuffer.push(line)
}

function writeLog() {
  fs.writeFileSync(LOG_FILE,
    '# Pooly Auto-Test Log\n' +
    'Generated: ' + new Date().toISOString() + '\n\n' +
    logBuffer.join('\n'))
}

function recordHistory(testKey, entry) {
  if (!fixHistory[testKey]) fixHistory[testKey] = []
  fixHistory[testKey].push({
    ...entry,
    timestamp: new Date().toISOString()
  })
}

function getHistory(testKey) {
  return fixHistory[testKey] || []
}

function wasAlreadyTried(testKey, fixDescription) {
  return getHistory(testKey)
    .some(h => h.fixApplied === fixDescription)
}

async function sleep(ms) {
  return new Promise(r => setTimeout(r, ms))
}

async function checkServerUp() {

      { timeout: 5000 })

}

async function restartServer() {
  log('Restarting server...')
  const { exec } = require('child_process')
  await new Promise(resolve => {
    exec("pkill -f 'node server.js'", () => resolve())
  })
  await sleep(1000)
  const { spawn } = require('child_process')
  const child = spawn('node', ['server.js'], {
    detached: true,
    stdio: 'ignore',
    cwd: '/home/runner/workspace'
  })
  child.unref()
  await sleep(3000)
  const up = await checkServerUp()
  if (!up) {
    log('SERVER FAILED TO RESTART')
    return false
  }
  log('Server restarted successfully')
  return true
}

async function runFix(fixNumber, fixName, fixFn) {
  log('\n--- FIX ' + fixNumber + ': ' + fixName + ' ---')
  let attempts = 0
  while (attempts < MAX_ATTEMPTS) {
    attempts++
    try {
      const result = await fixFn(attempts)
      if (result.success) {
        log('✅ FIX ' + fixNumber + ' PASSED: ' + fixName)
        writeLog()
        return true
      } else {
        log('❌ FIX ' + fixNumber + ' attempt ' +
          attempts + ' failed: ' + result.error)
        if (attempts >= MAX_ATTEMPTS) {
          log('🛑 FIX ' + fixNumber +
            ' EXHAUSTED 5 ATTEMPTS — stopping script')
          log('Recommended manual steps: ' + result.manualFix)
          writeLog()
          process.exit(1)
        }
        await sleep(1000)
      }
    } catch (err) {
      log('💥 FIX ' + fixNumber +
        ' exception: ' + err.message)
      if (attempts >= MAX_ATTEMPTS) {
        writeLog()
        process.exit(1)
      }
    }
  }
}

async function runTest(testNumber, testName, testFn) {
  log('\n=== TEST ' + testNumber + ': ' + testName + ' ===')
  const testKey = 'test-' + testNumber
  let attempts = 0

  while (attempts < MAX_ATTEMPTS) {
    attempts++
    log('--- Attempt ' + attempts + ' of ' +
      MAX_ATTEMPTS + ' ---')
    try {
      const result = await testFn(attempts, testKey)

      if (result.pass) {
        log('✅ PASS: ' + testName)
        log('Response: ' +
          JSON.stringify(result.data, null, 2))
        recordHistory(testKey, {
          attempt: attempts,
          error: null,
          hypothesis: null,
          fixApplied: null,
          result: 'pass'
        })
        writeLog()
        return result
      }

      log('❌ FAIL attempt ' + attempts + ': ' +
        result.reason)
      log('HTTP Status: ' + result.status)
      log('Response: ' +
        JSON.stringify(result.data, null, 2))
      log('Matched predicted cause: ' +
        (result.matchedCause || 'no'))

      recordHistory(testKey, {
        attempt: attempts,
        error: result.reason,
        hypothesis: result.hypothesis || 'unknown',
        fixApplied: result.fixApplied || 'none',
        result: 'fail'
      })

      if (attempts >= MAX_ATTEMPTS) {
        log('🛑 EXHAUSTED ' + MAX_ATTEMPTS +
          ' ATTEMPTS on TEST ' + testNumber)
        log('Full history for this test:')
        log(getHistory(testKey))
        log('RECOMMENDED MANUAL FIX:')
        log(result.manualFix ||
          'Review test-log.md for full diagnosis')
        writeLog()
        process.exit(1)
      }

      if (result.diagnoseAndFix) {
        log('Running diagnoseAndFix...')
        await result.diagnoseAndFix()
      }

      log('Retrying test ' + testNumber + '...')
      await sleep(2000)

    } catch (err) {
      log('💥 Exception in test ' + testNumber +
        ': ' + err.message)
      log(err.stack)
      recordHistory(testKey, {
        attempt: attempts,
        error: err.message,
        hypothesis: 'uncaught exception',
        fixApplied: 'none',
        result: 'fail'
      })
      if (attempts >= MAX_ATTEMPTS) {
        log('🛑 STOPPING — unrecoverable exception')
        writeLog()
        process.exit(1)
      }
      await sleep(2000)
    }
  }
}
